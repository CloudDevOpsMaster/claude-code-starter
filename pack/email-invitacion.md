## CUERPO DEL EMAIL

---

Hola,

¿Cuántas horas de tu semana se van en tareas que podrías delegar?

Escribir tests, crear endpoints, hacer code reviews, configurar pipelines de CI/CD… Son tareas necesarias. Pero no tienen que ser tuyas.

**Claude Code es un agente de desarrollo que vive en tu terminal.** Le das un objetivo, él analiza tu código, hace un plan, escribe los cambios, corre los tests y te pide aprobación. Tú eliges qué aprobar.

No es un copilot que autocompleta líneas. Es un colaborador que entiende tu proyecto completo.

---

### Lo que construirás en la sesión (90 minutos)

```
[INSTALAR]  →  [CONFIGURAR]  →  [PRIMER PROMPT]  →  [FEATURE COMPLETA]
   5 min           10 min            15 min               60 min
```

Al terminar tendrás:

✅ Claude Code instalado y configurado en tu máquina  
✅ Un `CLAUDE.md` personalizado para tus proyectos  
✅ Tu primer feature implementada por el agente (código + tests)  
✅ El repo sandbox listo para seguir practicando solo  

---

### El roadmap completo

**Bloque 1 — Setup (15 min)**
- Instalar Claude Code CLI
- Conectar tu API key
- Entender modos de permisos (interactivo vs headless)
- Crear tu primer `CLAUDE.md`

**Bloque 2 — Primer uso real (20 min)**
- Clonar el repo sandbox del curso
- Primer prompt: agregar una feature con `/plan`
- Ver cómo el agente planea, ejecuta y pide aprobación
- Iteración: cómo dar feedback y corregir rumbo

**Bloque 3 — Práctica guiada (55 min)**

| Ejercicio | Qué aprenderás |
|-----------|---------------|
| 01 · Sort por prioridad | `/plan` + aprobación de cambios |
| 02 · Generar tests | Skills + cobertura automática |
| 03 · Code review | El agente como reviewer de código con bugs |
| 04 · Feature completa | Endpoint + validación + tests end-to-end |

---

### Recursos que recibes

📁 **`claude-code-starter`** — Repo sandbox con API funcional, tests listos y frontend  
📄 **Guía rápida** — Todos los comandos clave en 1 página  
📄 **20 prompts listos** — Cópialos directamente en tus proyectos  
📄 **CLAUDE.md template** — Plantilla lista para cualquier stack  

**Instalación previa (2 minutos):**
```bash
npm install -g @anthropic-ai/claude-code
```

---

### ¿Para quién es esto?

→ Desarrolladores con al menos 6 meses de experiencia  
→ Equipos que quieren reducir tiempo en tareas repetitivas  
→ Tech leads que buscan estandarizar cómo el equipo usa IA  

No necesitas haber usado Claude Code antes. Solo necesitas una laptop y ganas de experimentar.

---

### Una cosa más

La mayoría de cursos de IA te enseñan prompts de ChatGPT.

Este te enseña a tener un agente que vive en tu repositorio, conoce tus convenciones, y trabaja contigo — no en lugar de ti.

La diferencia importa.
